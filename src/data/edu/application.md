# 學程申請

依申請人背景與目標學程不同，四大主要路徑：

## 1. 學院預備班 · Studienkolleg

**適用**：學歷未達德國大學直接入學門檻者（如台灣高中畢業無大學學歷、或大學未讀滿一定學期）。

**課程結構**：
- 一年制、五種課程方向（T / M / W / G / S）
- T 課程對應理工、M 對應醫學、W 對應經濟、G 對應人文、S 對應社會
- 結業考 Feststellungsprüfung（FSP）通過後才可申請大學

**申請注意**：
- 學費：公立多為免費、私立每年 €5,000-15,000
- 需先通過語言測試（多要求 B2 以上）
- Uni-assist 統一遞件

## 2. 學士 · Bachelor

**四種入學管道**：
- 直接申請（高中含大學基礎年）
- Studienkolleg + FSP
- 已有台灣大學一年級 + 良好成績
- APS 審核通過 + 語言證明

**語言門檻**：
- DSH-2 / TestDaF 4x4 / telc C1 Hochschule 三選一
- 部分英語授課學程需 IELTS 6.5 / TOEFL 90+

**申請時程**（冬季學期）：
1. 4-5 月 Uni-assist 開放
2. 7 月中截止（各校不同）
3. 8 月放榜
4. 10 月開學

## 3. 碩士 · Master

**基本要求**：
- 相關領域學士學位
- 語言證明（多數為 IELTS 6.5 / TOEFL 90 或 DSH-2）
- 動機信 + CV + 推薦信

**MBA / 特殊學程**：可能需 GMAT / GRE、工作經驗、面試。

**熱門學程**：
- TU München / TU Berlin 工程類
- LMU / Heidelberg 人文與醫科
- Frankfurt / Mannheim 商管
- WHU / ESMT 商學院（私立）

## 4. 博士 · PhD

**兩種模式**：
- **結構化 PhD 計畫**（Doctorate Program）：類似碩士流程、有課程、3-4 年
- **個人導向 PhD**（Individual）：先找教授、寫 proposal、教授同意即可

**經費**：
- DAAD 獎學金
- 教授研究計畫（Wissenschaftlicher Mitarbeiter, 半薪或全薪）
- 台灣教育部公費留學

## APS 審核

台灣、中國、越南等特定國家申請人**必須**先過 APS 學歷審核，才能申請德國大學。可經 Zeugnisbewertung 或面試 APS。
