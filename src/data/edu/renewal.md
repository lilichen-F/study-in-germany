# 延簽流程 · Aufenthaltstitel

簽證是進德國的入場券、居留許可（Aufenthaltstitel）才是留下來的憑證。落地 90 天內必須到 Ausländerbehörde 辦第一次居留許可、之後每 1-2 年續簽一次。

## 第一次辦理 · 落地 90 天內

**準備文件**：
- 護照 + 現行簽證
- Anmeldung 證明
- 語校 / 大學註冊證明
- 財力證明（Sperrkonto 餘額 + 次年計畫）
- 醫療保險證明
- 生物特徵費 €100（現場繳）
- 護照照片（生物特徵規格）

**流程**：
1. 線上預約 Termin（大城市可能等 2-4 個月）
2. 準備所有文件 + 影本
3. 面談（DE / EN 混用可）
4. 錄指紋 + 拍照
5. 3-6 週後拿卡

**注意**：若簽證過期而 Termin 尚未到，需申請「Fiktionsbescheinigung」（過渡文件），到 Ausländerbehörde 現場排。

## 續簽 · 效期屆滿前 4-8 週

**與第一次差別**：
- 需證明**財力持續**：新的 Sperrkonto 或 Nachweis
- 需證明**學業進度**：語校成績單 / 大學已修學分
- 有工作者：勞動合約 + 稅單

**續簽被拒常見原因**：
- 語校未達每週 18 小時上課時數
- 大學連續兩學期未修學分
- 保險保單無涵蓋居留全期
- 財力不足

## 學業轉工作 · Bluecard / 工作居留

大學畢業後可申請「求職簽證」18 個月（Arbeitsplatzsuche），或直接持工作合約申請 Bluecard（薪資門檻 2024/25 為 €45,300/年，STEM 領域 €41,041/年）。

Bluecard 兩個關鍵優勢：33 個月後可申請永久居留；21 個月後若 B1 通過也可申請永居。

## 學業轉自僱 · Freiberufler

適用於自由工作者（設計、翻譯、藝術等）。需商業計畫、財力、稅務認證。難度較高、成功率隨城市與行業差異大。
