# 落地指南

前 30 天最忙的行政事項。順序有優先級：**住宿 → Anmeldung → 銀行 → 保險 → SIM → 電視稅**。前三項必須在落地兩週內完成，否則影響後續所有事。

## 1. 住宿確認 · Wohnung / WG

落地前若能簽好正式租約最佳（Nachmieter 或 Erstbezug）。無法者，先訂 hostel 或 Airbnb 兩週作為過渡，同時：
- 上 wg-gesucht.de / immobilienscout24.de 找長租
- 準備 SCHUFA（信用紀錄，剛到無紀錄可用父母匯款證明或雇主證明替代）

**警告**：從 wg-gesucht 找房遇「請先匯定金再看房」訊息 100% 詐騙，直接封鎖。

## 2. Anmeldung · 戶籍登記

落地 14 天內必須到當地 Bürgeramt 辦。**沒有 Anmeldung 拿不到稅號、開不到帳戶、辦不了保險。**

**準備文件**：
- 護照
- 租約（Mietvertrag）
- 房東確認書（Wohnungsgeberbestätigung）
- Anmeldung 表單（現場拿或線上填）

**注意**：柏林 / 慕尼黑等大城市 Termin 難搶，落地前一週就要開始盯線上預約系統。

## 3. 銀行帳戶 · Konto

**選項對比**：
- **傳統銀行**（Sparkasse / Volksbank）：Bar deposit 方便、少數房東要求本地帳戶、辦事需 Termin
- **Neobank**（N26 / Revolut / Vivid）：手機 App 開戶、5 分鐘完成、國際轉帳快

建議：Neobank 為主帳戶（收薪 / 日常）、Sparkasse 為備用（現金業務）。

## 4. 保險 · Krankenversicherung

**兩大類**：
- **法定 GKV**（TK / AOK / Barmer / DAK 等）：學生 €120 前後 / 月、正式學生身分才能保
- **私人 PKV**（Mawista / Care Concept 等）：語校生前期唯一選項、€30-60 / 月、續簽時保單需完整涵蓋

**建議**：語校期間 PKV → 進大學正式註冊後轉 GKV。

## 5. SIM 卡 · Mobilfunk

- **Prepaid**：Aldi Talk / Lidl Connect / Vodafone Callya 現場超市買、€10-20 / 月
- **後付**：需 SCHUFA + 銀行帳戶、綁 24 個月、O2 / Vodafone / Telekom 三家

短期建議 Prepaid、住定後再考慮後付。

## 6. Rundfunkbeitrag · 電視稅

住定後每戶每月 €18.36（2024/25），不繳會罰。學生無法免除但可分攤：一戶只繳一份。到 rundfunkbeitrag.de 註冊。

## 官方參考
- [wg-gesucht.de 找 WG](https://www.wg-gesucht.de/)
- [Bürgeramt 預約](https://service.berlin.de/)（各邦不同）
- [Rundfunkbeitrag](https://www.rundfunkbeitrag.de/)

> baseline 版本，各邦執行細節與費用略有差異，正式辦理前請查詢所在邦官方公告。
